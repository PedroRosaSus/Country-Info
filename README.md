# Country Info

## O Country info é um programa que usa de base a API RestCountries para buscar informações de países, com uma fácil vizualização ao usuário <br />
## Ele mostra as seguintes informações de um país:<br />
    - Nome comum
    - Nome oficial 
    - Capital 
    - Língua 
    - Região
    - População
    - Moeda 
    - Símbolo
